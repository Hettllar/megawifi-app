# نظام حماية وتحكم IPTV

تم إضافة نظام أمان متعدد المستويات لخدمة IPTV يوفر تحكماً كاملاً للأدمن.

## الميزات الرئيسية

### 1. التفعيل اليدوي من الأدمن
- **لا يتم تفعيل IPTV تلقائياً** للمشتركين الجدد
- يجب على الأدمن تفعيل الخدمة يدوياً لكل مشترك
- زر تفعيل/إلغاء تفعيل في صفحة المشترك
- إنشاء اشتراك IPTV تلقائياً عند التفعيل الأول

### 2. حصر نطاق IP
يمكن تحديد نطاق IP محدد لكل مشترك بثلاث طرق:

#### أ. IP محدد (Exact IP)
```
192.168.1.100
```
السماح فقط من هذا الـ IP بالضبط

#### ب. نطاق بالنجمة (Wildcard)
```
192.168.1.*
```
السماح لجميع IPs في النطاق 192.168.1.0 - 192.168.1.255

#### ج. نطاق CIDR
```
10.0.0.0/24
```
السماح للنطاق 10.0.0.0 - 10.0.0.255

#### د. عدة نطاقات
```
192.168.1.*, 10.0.0.0/24, 172.16.0.1
```
افصل بين النطاقات بفاصلة

### 3. رسائل الخطأ الواضحة
- `IPTV service is not enabled` - الخدمة غير مفعلة
- `Access denied from this IP` - IP غير مسموح
- `Invalid credentials` - بيانات خاطئة
- `Expired subscription` - اشتراك منتهي

## كيفية الاستخدام

### لتفعيل IPTV لمشترك:

1. اذهب إلى صفحة المشترك: `المشتركين > اختر مشترك`
2. في الشريط الجانبي، اضغط على زر **"تفعيل IPTV"**
3. سيتحول الزر إلى بنفسجي ويظهر قسم إعدادات IPTV
4. سيتم إنشاء اشتراك IPTV تلقائياً بـ:
   - Username: `iptv_sub_{subscriber_id}`
   - Password: عشوائي 12 حرف
   - صلاحية: سنة واحدة
   - عدد الاتصالات: 2 جهاز

### لتحديد نطاق IP:

1. في قسم إعدادات IPTV (يظهر فقط عند التفعيل)
2. أدخل النطاق في حقل "حصر الـIP"
3. اضغط **"حفظ إعدادات IP"**

### للمشترك (صفحة استعلام الرصيد):

1. يفتح: https://megawifi.site/check-balance
2. يدخل رقم هاتفه
3. إذا كان IPTV مفعل، سيظهر قسم قابل للطي
4. يمكنه:
   - تحميل VLC Player
   - تحميل ملف M3U المخصص له
   - قراءة تعليمات الاستخدام

```

```

### مثال 2: السماح من عدة مواقع
```

```

### مثال 3: بدون قيود (خطر!)
اترك الحقل فارغاً - سيسمح من أي IP (غير مستحسن)

## ملاحظات أمنية

⚠️ **تحذيرات:**
- احذر من ترك حصر IP فارغاً (يسمح من أي مكان)
- استخدم CIDR للشبكات الكبيرة
- راقب سجلات الوصول بانتظام

✅ **أفضل الممارسات:**
- فعّل IPTV فقط للمشتركين الموثوقين
- حدد نطاق IP ضيق قدر الإمكان
- راجع الاشتراكات شهرياً
- عطّل IPTV للمشتركين المنتهية اشتراكاتهم
