تم إضافة خيارات IPTV إلى صفحة UserManager! ✅

## الخيارات المتاحة الآن:

### في صفحة المشترك (UserManager):

#### 1. زر تفعيل/إلغاء IPTV 🟣
- **الموقع:** في قسم "الإجراءات" (الشريط الجانبي)
- **اللون:** بنفسجي عند التفعيل، رمادي عند الإلغاء
- **الوظيفة:** تفعيل أو إلغاء خدمة IPTV للمشترك
- **ميزة:** يظهر في Desktop وMobile

#### 2. قسم إعدادات IPTV 📺
يظهر تلقائياً عند تفعيل IPTV ويحتوي على:

**أ. معلومات الاشتراك:**
- اسم المستخدم (IPTV Username)
- كلمة المرور (مع زر إظهار/إخفاء 👁️)

**ب. حصر نطاق IP:**
- حقل إدخال لتحديد IPs المسموحة
- دعم 3 صيغ:
  * IP محدد: `192.168.1.100`
  * Wildcard: `192.168.1.*`
  * CIDR: `10.0.0.0/24`
  * متعدد: `192.168.1.*, 10.0.0.0/24`
- زر "حفظ إعدادات IP" 💾

**ج. رابط M3U:**
- عرض الرابط الكامل
- زر نسخ سريع (اضغط على الرابط)
- تنبيه "تم النسخ!" ✅

## طريقة الاستخدام:

### للأدمن:

1. **افتح UserManager:**
   - اذهب إلى: قائمة ← UserManager
   - اختر أي مشترك من القائمة

2. **تفعيل IPTV:**
   - في الشريط الجانبي (أو قسم الإجراءات في الموبايل)
   - اضغط زر "تفعيل IPTV"
   - سيصبح اللون بنفسجي
   - ستظهر رسالة: "تم تفعيل IPTV بنجاح"
   - الصفحة ستتحدث تلقائياً

3. **قسم إعدادات IPTV سيظهر:**
   - بطاقة باللون البنفسجي الفاتح
   - تحتوي على جميع المعلومات

4. **تحديد نطاق IP (اختياري):**
   - في حقل "حصر الـIP"
   - أدخل IP أو نطاق (مثال: `192.168.1.*`)
   - اضغط "حفظ إعدادات IP"
   - ستظهر رسالة: "تم حفظ إعدادات IP بنجاح" ✅

5. **نسخ رابط M3U:**
   - اضغط على الرابط الطويل في قسم "رابط M3U"
   - سيتم نسخه تلقائياً
   - سيظهر "تم النسخ!" لمدة ثانيتين

### للمشترك:

- يفتح صفحة استعلام الرصيد
- يدخل رقم هاتفه
- إذا IPTV مفعل، سيظهر القسم المنسدل
- يحمل VLC + ملف M3U

## الفرق بين Subscribers و UserManager:

| الميزة | Subscribers | UserManager |
|--------|-------------|-------------|
| نوع الاتصال | PPP/PPPoE/Hotspot | UserManager |
| الصفحة | `/subscribers/{id}` | `/usermanager/{id}` |
| خيارات IPTV | ✅ متوفرة | ✅ متوفرة (الآن!) |
| التحكم بالـIP | ✅ | ✅ |
| نسخ M3U | يدوي | بزر! 📋 |

## اختبار الميزة:

### خطوة بخطوة:

1. افتح UserManager في المتصفح:
   ```
   https://megawifi.site/usermanager
   ```

2. اختر أي مشترك من القائمة

3. في صفحة المشترك:
   - ابحث عن قسم "الإجراءات" (يمين الشاشة)
   - يجب أن ترى زر "تفعيل IPTV" بلون رمادي

4. اضغط "تفعيل IPTV":
   - اللون سيتحول إلى بنفسجي
   - الصفحة ستتحدث
   - سيظهر قسم "إعدادات IPTV" جديد

5. في قسم "إعدادات IPTV":
   - جرب إظهار/إخفاء كلمة المرور
   - أدخل نطاق IP واحفظ
   - اضغط على رابط M3U لنسخه
